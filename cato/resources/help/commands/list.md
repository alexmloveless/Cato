# /list

Show lists and list items.

## Usage

```
/list                    # Show all lists with item counts
/list shopping           # Show items in shopping list
```

## Arguments

- `list_name` (optional) - Name of list to show items for

## Examples

```
/list                    # Show all lists
/list shopping           # Show shopping list items
/list ideas              # Show ideas list items
```

## Aliases

- `/lists`
- `/show-lists`
